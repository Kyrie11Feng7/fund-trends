# 科技基金趋势分析 · 前瞻信号层设计（可集成版）

> 目标：在你现有"宏观→情绪→估值→个基→新闻"四层框架上，补一层**前瞻信号**，把"描述"升级为"预判"。
> 数据基准：2026-07-17 收盘（美股底层指数 + A 股可交易代理的技术面）。

---

## 1. 信号层解决什么问题

你现在的站点是**优秀的描述型面板**。信号层要回答用户最关心的两件事：
1. **短期（1–4 周）这笔基金动能如何、有没有反弹条件？**
2. **中期（1–3 月）它处在上行趋势还是已经转弱？**

输出不是"买/卖"建议，而是**可解释的标签 + 0–100 动能/结构分 + 风险等级**，让用户自己决策。

---

## 4. 信号规则（可复现）

### 短期动能分（0–100，越高越接近反弹/走强）
```
oversold  = rsi6 < 35 且 close <= bollLower
belowShort= close < ma20 且 macd < 0
strongTrend = adx > 25

if oversold 且 !strongTrend : 标签"超卖待反弹"  score≈38  反弹概率高
if oversold 且  strongTrend : 标签"超卖但趋势强" score≈30  反弹易中继
if !belowShort 且 macd>0    : 标签"偏强"        score≈70
else                         : 标签"弱势下行"    score≈30
```

### 中期结构分（0–100，越高趋势越健康）
```
if close > ma120 且 close > ma250 : "长牛结构未破"  score≈78
elif close > ma250                : "中性"          score≈55
else                              : "趋势转弱"      score≈35
```

### 风险等级
- 高：G2（半导体 ADX=34，强跌势，弹性最大）
- 中低：G1（回撤最浅、长牛最完整）
- 中：G3 / G4 / G5

## 5. 前端渲染建议（"信号卡"）

在「基金总览」每行加三列或一个角标：
`[短期:超卖待反弹 38] [中期:长牛修正 80] [风险:中低]`
并用颜色区分：红=弱势、黄=超卖反弹、绿=偏强。

在页面顶部加一个**分组概览**：5 组的短期/中期信号 + 最强/最弱组高亮，让用户一眼看懂"现在科技基金整体处在什么阶段"。

## 6. 信号计算函数（可直接放进你的 pipeline）

```js
function computeTrendSignal(t) {
  const oversold   = t.rsi6 < 35 && t.close <= t.bollLower;
  const belowShort = t.close < t.ma20 && t.macd < 0;
  const strongTrend= t.adx > 25;

  let shortTerm, sScore;
  if (oversold && !strongTrend)      { shortTerm = '超卖待反弹'; sScore = 38; }
  else if (oversold && strongTrend)  { shortTerm = '超卖但趋势强'; sScore = 30; }
  else if (!belowShort && t.macd>0)  { shortTerm = '偏强'; sScore = 70; }
  else                               { shortTerm = '弱势下行'; sScore = 30; }

  let midTerm, mScore;
  if (t.close > t.ma120 && t.close > t.ma250) { midTerm = '长牛结构未破'; mScore = 78; }
  else if (t.close > t.ma250)                 { midTerm = '中性'; mScore = 55; }
  else                                        { midTerm = '趋势转弱'; mScore = 35; }

  return { shortTerm, sScore, midTerm, mScore };
}
// 用法：computeTrendSignal({close:695.33, ma20:718.30, ma60:709.48, ma120:657.36, ma250:626.38, macd:-5.63, rsi6:30.70, kdjJ:19.68, adx:15.62, bollLower:695.54})
```
